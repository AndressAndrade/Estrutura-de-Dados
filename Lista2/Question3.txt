3) Suponha que os elementos do vetor v são do tipo int e cada int ocupa 8 bytes no seu computador. Se o endereço de v[0] é 55000, qual o valor da expressão v + 3?

Num vetor, o endereço dos elementos é sequencial, se o endereço de v[0] é 55000 e cada int ocupa 8 bytes.

&v[0] = 55000
&v[1] = 55008 
&v[2] = 55016
&v[3] = 55024

O valor da expressão v + 3 = 55024
